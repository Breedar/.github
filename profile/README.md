# Breedar

We build open-source infrastructure for global payroll, identity, and cross-border finance on the Stellar network.

## Projects

| Project | Description |
|---|---|
| [paystream](https://github.com/Breedar/paystream) | Global payroll on Stellar. Batched stablecoin payouts to 1,000 recipients, settled in 47 local currencies via SEP-31. |
| [StellarRemit](https://github.com/Breedar/StellarRemit) | Cross-border remittance rails built on Stellar anchors. |
| [StellarID](https://github.com/Breedar/StellarID) | Decentralized identity and KYC attestation for Stellar accounts. |
| [PayVault](https://github.com/Breedar/PayVault) | Multi-sig treasury management for teams and DAOs on Stellar. |
| [AssetsUp](https://github.com/Breedar/AssetsUp) | Tokenized asset issuance and lifecycle management on Stellar. |

## Stack

Go, TypeScript, React, PostgreSQL, Stellar/Horizon, SEP-10/SEP-31.
